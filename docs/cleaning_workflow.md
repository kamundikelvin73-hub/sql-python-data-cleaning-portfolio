# Data Cleaning Workflow

## Step 1: Inspect Dataset

Review columns, missing values, duplicates, and inconsistent categories.

## Step 2: Standardize Values

Clean emails, category names, and status values.

## Step 3: Validate Dataset

Check required fields, duplicate IDs, and invalid values.

## Step 4: Generate Report

Summarize cleaning actions and remaining issues.

## Step 5: Prepare for AI/ML Use

Ensure the dataset is structured, consistent, and documented.
